# Engineering Automation & CAD Customization Portfolio

This repository showcases clean, generic implementations of automation tools designed for automotive engineering environments (e.g., Creo API, SolidWorks API, and MATLAB integration workflows). 

*Note: All code here consists of generic functional design patterns built from scratch to respect corporate non-disclosure agreements (NDAs).*

---

## Project Showcase

### 1. DIN-Compliant Bill of Materials (BOM) Automation Engine
* **Directory:** `/bom_automation/`
* **Language:** Python 3.x
* **Core Functionality:** Simulates parsing raw parametric structural metadata directly from a CAD database. It validates engineering attributes and structures data into a localized, German-industry standard semicolon-delimited (`;`) CSV format ready for ERP systems like SAP Business One.
* **Key Skills Shown:** Data normalization, file I/O operations, industrial standards application (DIN/ISO).
